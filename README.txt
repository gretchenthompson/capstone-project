# Massage-Therapy-Site

This is a website for my massage business.
The website is a place for clients to find me, get to know me,
get to know the benefits of massage and contact me.
It will link them to an online scheduling tool.

## CSS features
- Flexbox to change from column in mobile view to row wrap in tablet view
- nth child to rearrange content on desktop view
- Fixed header on page scroll

## JS features
- Toggle function to show/hide definition of a defining moment
- Recursive function to slide thru multiple client reviews

Verified functionality in Chrome, Firefox and Edge.

WAVE(web accessibility evaluation tool powered by WebAIM) shows no errors.

Images from unsplash.com license states all photos can be downloaded and used for free.
